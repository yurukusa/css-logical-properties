How It Works

Logical Properties

CSS logical properties use inline (text direction) and block (block layout direction) instead of physical directions (top/right/bottom/left). They automatically adapt to writing mode and text direction.

Why Use Logical Properties?

• Internationalization: Auto-adapts for RTL languages
• Writing modes: Works with vertical text
• Consistency: One set of properties for all layouts
• Shorthands: margin-inline, padding-block (2 values in 1)
• Future-proof: CSS is moving toward logical by default
